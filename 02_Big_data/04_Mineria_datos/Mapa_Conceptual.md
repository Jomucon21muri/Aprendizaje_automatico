# Mapa Conceptual: Minería de Datos
## Big Data - Bloque 4

### Proceso de Minería de Datos (CRISP-DM)

```
├── 1. Business Understanding
│   ├── Definir objetivos
│   ├── Identificar stakeholders
│   └── Traducir a objetivos técnicos
├── 2. Data Understanding
│   ├── Exploración inicial
│   ├── Estadísticas descriptivas
│   └── Identificar problemas de datos
├── 3. Data Preparation
│   ├── Limpieza
│   ├── Integración
│   ├── Transformación
│   └── Feature engineering
├── 4. Modeling
│   ├── Selección de técnica
│   ├── Diseño de experimentos
│   ├── Entrenamiento
│   └── Evaluación
├── 5. Evaluation
│   ├── Evaluar resultados
│   ├── Revisar proceso
│   └── Decisión: Deploy o iterar
└── 6. Deployment
    ├── Integración en producción
    ├── Monitoreo
    └── Mantenimiento
```

### Técnicas de Minería

```
Minería de Datos
    ├── Predictiva
    │   ├── Clasificación
    │   ├── Regresión
    │   └── Forecasting
    ├── Descriptiva
    │   ├── Clustering
    │   ├── Asociación
    │   └── Outlier detection
    └── Prescriptiva
        ├── Optimización
        └── Recomendaciones
```

### Métricas de Éxito
- ROI (Retorno de Inversión)
- Precisión/Recall del modelo
- Reducción de costos
- Aumento de ingresos
- Mejora operacional

### Desafíos
- Calidad de datos
- Escalabilidad
- Privacidad y cumplimiento
- Mantenimiento de modelos
- Explicabilidad
