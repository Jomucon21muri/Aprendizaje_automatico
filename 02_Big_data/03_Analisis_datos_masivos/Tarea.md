# Tarea: análisis de datos masivos
## Big Data - Bloque 3

### Objetivo
Realizar análisis completo de dataset masivo aplicando técnicas de Big Data.

### Actividades

#### Actividad 1: selección y preparación dataset
- Selecciona dataset grande (>500MB) o genera sintético
- Entiende estructura y significado
- Carga en ambiente distribuido (Spark)
- Realiza limpieza inicial

#### Actividad 2: EDA a escala
- Estadísticas descriptivas distribuidas
- Análisis de distribuciones
- Detección de valores faltantes
- Identificación de anomalías
- Correlaciones entre variables

#### Actividad 3: transformación y Feature Engineering
- Crea nuevas características relevantes
- Normaliza datos
- Codifica variables categóricas
- Maneja desbalances si aplica

#### Actividad 4: análisis ML distribuido
Implementa al menos 2 modelos:
- Regresión o Clasificación
- Clustering
Aplica validación cruzada, optimiza hiperparámetros.

#### Actividad 5: visualización e insights
- Crea gráficos interpretables
- Genera dashboard interactivo
- Documenta hallazgos principales
- Propone acciones basadas en insights

#### Actividad 6: rendimiento y escalabilidad
- Compara tiempo ejecución (local vs distribuido)
- Analiza overhead de distribución
- Propone optimizaciones
- Documenta lecciones aprendidas

### Criterios de evaluación
- Preparación dataset: 15%
- EDA distribuido: 20%
- Feature engineering: 15%
- Análisis ML: 20%
- Visualización e insights: 20%
- Análisis rendimiento: 10%

### Entrega
- Notebook Spark comentado
- Dataset procesado
- Gráficos y visualizaciones
- Dashboard interactivo (Jupyter widgets o Tableau)
- Reporte ejecutivo con recomendaciones
