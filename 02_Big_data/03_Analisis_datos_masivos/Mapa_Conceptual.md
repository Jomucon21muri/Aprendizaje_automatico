# Mapa conceptual: análisis de datos masivos
## Big Data - Bloque 3

### Pipeline de análisis

```
Datos Masivos
    ↓
Ingesta (ETL/ELT)
    ↓
Limpieza y Transformación
    ↓
Almacenamiento (Data Warehouse/Lake)
    ↓
Análisis (SQL, Stats, ML)
    ↓
Visualización y Insights
    ↓
Decisiones Accionables
```

### Técnicas de análisis

**Estadístico**
- Descriptiva en escala masiva
- Sampling estratificado
- Correlaciones distribuidas

**Machine Learning distribuido**
- Regresión escalable
- Clustering en paralelo
- Feature engineering masivo

**Análisis exploratorio (EDA)**
- Perfiles de datos
- Detección de anomalías
- Relaciones entre variables

### Desafíos especiales
- Escalabilidad y distribución
- Manejo de valores faltantes
- Sesgo en muestreo
- Interpretabilidad con muchas features

### Herramientas
- SQL distribuido (Spark SQL, Presto)
- BI Tools (Tableau, Looker, Kibana)
- Jupyter + PySpark
- Datadog, Grafana (monitoreo)
