# Mapa conceptual: deep learning
## Sistemas de aprendizaje automático - Bloque 4

### Arquitecturas de deep learning

```
Deep Learning
    ├── Redes Convolucionales (CNN)
    │   ├── Convolución 1D/2D/3D
    │   ├── Pooling
    │   └── Aplicaciones: visión, detección objetos
    ├── Redes Recurrentes (RNN)
    │   ├── LSTM (Long Short-Term Memory)
    │   ├── GRU (Gated Recurrent Unit)
    │   └── Aplicaciones: series temporales, NLP
    ├── Autoencoders
    │   ├── Autoencoders simples
    │   ├── Variational AE (VAE)
    │   └── Aplicaciones: compresión, generación
    ├── Redes Generativas
    │   ├── GAN (Generative Adversarial Networks)
    │   ├── Diffusion Models
    │   └── Aplicaciones: síntesis de imágenes
    └── Arquitecturas Modernas
        ├── ResNet, DenseNet
        ├── Inception, EfficientNet
        └── MobileNet, SqueezeNet
```

### Componentes clave
- Capas de convolución (Conv, MaxPool, BatchNorm)
- Funciones de activación avanzadas (ReLU, Leaky ReLU, ELU)
- Técnicas de regularización (Dropout, L1/L2)
- Optimizadores modernos (Adam, RMSprop, SGD)

### Técnicas de entrenamiento
- Transfer Learning
- Fine-tuning
- Data Augmentation
- Batch Normalization
- Regularización avanzada
