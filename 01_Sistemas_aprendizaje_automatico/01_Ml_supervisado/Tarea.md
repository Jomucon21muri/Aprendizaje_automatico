# Tarea: machine learning supervisado
## Sistemas de aprendizaje automático - Bloque 1

### Objetivo
Implementar y comparar múltiples algoritmos de ML supervisado en un dataset real.

### Actividades

#### Actividad 1: preparación de datos
- Carga un dataset (UCI, Kaggle, sklearn datasets)
- Realiza limpieza y preprocesamiento
- Normaliza/estandariza características
- Maneja valores faltantes
- Codifica variables categóricas

#### Actividad 2: implementación de algoritmos
Implementa al menos 4 modelos:
1. Regresión logística
2. Árbol de decisión
3. SVM
4. Random Forest

#### Actividad 3: entrenamiento y validación
- Divide datos (70% train, 30% test)
- Entrena cada modelo
- Aplica validación cruzada (5-Fold)
- Afina hiperparámetros

#### Actividad 4: comparación y análisis
- Calcula todas las métricas relevantes
- Crea tabla comparativa
- Grafica curvas ROC, matriz de confusión
- Identifica el mejor modelo
- Documenta conclusiones

### Criterios de evaluación
- Preparación de datos: 20%
- Implementación correcta: 30%
- Validación rigurosa: 25%
- Análisis comparativo: 25%

### Entrega
- Notebook Jupyter comentado
- Tabla comparativa de resultados
- Gráficos de evaluación
- Reporte con recomendaciones
