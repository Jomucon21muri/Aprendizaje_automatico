# Contenido teórico: Sistemas expertos
## Sistemas de aprendizaje automático - Bloque 8

## 1. Introducción a los sistemas expertos

### 1.1 Definición y propósito
Un Sistema Experto es un programa informático que contiene y utiliza el conocimiento de expertos en un campo específico para resolver problemas.

### 1.2 Ventajas
- Disponibilidad permanente del conocimiento experto
- Uniformidad en decisiones
- Reducción de costes operacionales
- Documentación del conocimiento
- Capacitación de nuevos usuarios

## 2. Arquitectura de sistemas expertos

### 2.1 Base de conocimiento
- Hechos: Información estática sobre el dominio
- Reglas: Relaciones causales (Si-Entonces)
- Heurísticas: reglas prácticas basadas en experiencia

### 2.2 Motor de inferencia
- Encadenamiento hacia adelante (Forward Chaining)
- Encadenamiento hacia atrás (Backward Chaining)
- Resolución de conflictos

### 2.3 Subsistema de explicación
Permite al usuario entender cómo se llegó a una conclusión.

## 3. Metodología de desarrollo

### 3.1 Fases
1. Identificación del problema
2. Adquisición del conocimiento
3. Representación del conocimiento
4. Implementación
5. Validación y pruebas
6. Mantenimiento

### 3.2 Roles
- Experto del dominio
- Ingeniero del conocimiento
- Programador
- Usuario final

## 4. Aplicaciones prácticas
- Diagnóstico médico
- Asesoramiento financiero
- Control de procesos industriales
- Análisis de riesgos
- Configuración de sistemas
