# Tarea: machine learning no supervisado
## Sistemas de aprendizaje automático - Bloque 2

### Objetivo
Aplicar técnicas no supervisadas para descubrir patrones en datos reales.

### Actividades

#### Actividad 1: dataset y exploración
- Selecciona dataset sin etiquetas o usa solo features
- Análisis exploratorio: distribuciones, correlaciones
- Identifica características relevantes
- Prepara datos: normalización, manejo de valores faltantes

#### Actividad 2: determinación de k óptimo
- Usa método Elbow
- Calcula Silhueta para valores de k
- Determina k recomendado
- Justifica tu elección

#### Actividad 3: comparación de algoritmos
Implementa al menos 3:
1. K-Means
2. Hierarchical Clustering
3. DBSCAN

Compara resultados con métricas intrínsecas.

#### Actividad 4: visualización y análisis
- Reduce dimensionalidad con PCA o t-SNE
- Visualiza clusters en 2D/3D
- Interpreta características de cada cluster
- Identifica anomalías

#### Actividad 5: documentación
- Describe patrones descubiertos
- Analiza usabilidad de cada algoritmo
- Propone aplicaciones prácticas

### Criterios de evaluación
- Preparación de datos: 15%
- Determinación de k: 15%
- Implementación: 30%
- Visualización y análisis: 25%
- Interpretabilidad: 15%

### Entrega
- Notebook Jupyter con código completo
- Visualizaciones (gráficos de clusters, PCA, t-SNE)
- Tabla comparativa de algoritmos
- Reporte de conclusiones
