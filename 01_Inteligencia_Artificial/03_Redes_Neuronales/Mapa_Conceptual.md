# Mapa Conceptual: Redes Neuronales
## Inteligencia Artificial - Bloque 3

### Estructura de una Neurona Artificial
```
Entradas (X1, X2, Xn)
         ↓
  Pesos (W1, W2, Wn)
         ↓
    Suma ponderada + Sesgo (b)
         ↓
    Función de Activación
         ↓
    Salida (Y)
```

### Tipos de Redes Neuronales
- **Perceptrón**: Red neuronal de una capa
- **Multicapa**: Redes con capas ocultas
- **Convolucionales (CNN)**: Para procesamiento de imágenes
- **Recurrentes (RNN)**: Para secuencias temporales
- **LSTM/GRU**: Para dependencias a largo plazo

### Componentes Clave
- Capa de entrada
- Capas ocultas
- Capa de salida
- Funciones de activación
- Optimizadores (SGD, Adam, etc.)
- Función de pérdida
